## ☃️ fdm-cli
freedom-cli ( 向往自由的脚手架 )

## 📜 目的
提高脚手架的自由度，可以灵活的搭建自己想要的初始化项目

🎁 学习如何设计和开发一款自己的脚手架
⭐ 磨练编程技能，学习优秀的编程实践
🎊 结识一群热爱学习、热爱开源的朋友
